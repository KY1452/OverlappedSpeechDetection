Klass Overlapped-Speech-Detection (OSD) Documentation

Overview

Klass OSD is a Python framework for training and serving a wavLM Model to detect overlapped speech detection. It is built to train a classification model that is specifically focused for audio applications within a meeting room setting.

Key Features

  • Kedro-managed pipelines for:

    • data processing to generate overlapped speech labels (ingesting Libri2Mix and SparseLibriMix datasets)

    • train set data segmentation

    • eval/test/sparselibrimix set data chunking

    • train set sampling and hashing

    • train data validation

    • curating huggingface datasets

    • model training

    • model prediction & evaluation

  • Scalable model serving using a FastAPI + Celery Docker-compose application
