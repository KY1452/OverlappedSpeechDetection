Environment Setup for Data Preparation & Model Training

Overview

This document covers the environment setup for the kedro pipeline which covers data preparation and model training.

Training Environment

Begin by downloading the klass-osd-kedro-pipeline.zip file and unzipping it. Navigate to the klass-osd folder.

Install Miniconda3

# Download the Miniconda installer for Linux (64-bit)
$ wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh -O minicond a_installer.sh

# Run the installer script
$ bash miniconda_installer.sh -b -p $ HOME/miniconda

# Add Miniconda to your PATH
$ echo 'export PATH="$HOME/miniconda/bin:$PATH"' >> $HOME/.bashrc
$ source  $HOME/.bashrc

# Verify the installation
$ conda --version

Set up Virtual Environment and Install Python Libraries

Create virtual environment:

$ conda env create -f klass2-conda-env.yaml
$ conda activate klass2

Install libsndfile-dev, librubberband-dev and rubberband-cli

$ sudo apt-get install libsndfile-dev librubberband-dev rubberband-cli
